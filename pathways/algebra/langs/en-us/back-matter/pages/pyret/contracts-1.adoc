= Contracts

Contracts tell us how to use a function. For example:  `ellipse {two-colons} (width +::+ Number, height +::+ Number, style +::+ String, color +::+ String) -> Image` tells us that the name of the function is  `ellipse`, it takes four inputs (2 Numbers & 2 Strings... more specifically width, height, solid/outline, color), and it evaluates to an  `Image`.
